Title: Markdown 备忘

Date: 2012-09-04 20:24:56

Markdown 是当前最流行也最简单易用的标记语法. 本博客的所有文章均使用 Markdown 写成.

### HEADERS

    This is an H

    =============

    This is an H

    -------------

    # This is an H1

    ## This is an H2

    ###### This is an H

    `

    ### BLOCKQUOTES

    `&gt; This is a blockquote with two paragraphs. Lorem ipsum dolor sit amet

    consectetuer adipiscing elit. Aliquam hendrerit mi posuere lectus

    Vestibulum enim wisi, viverra nec, fringilla in, laoreet vitae, risus.

    &gt; Donec sit amet nisl. Aliquam semper ipsum sit amet velit. Suspendiss

    id sem consectetuer libero luctus adipiscing

    `

    ### LISTS

    `*   Re

    *   Gree

    *   Blue

    +   Re

    +   Gree

    +   Blue

    -   Re

    -   Gree

    -   Blue

    1.  Bir

    2.  McHal

    3.  Paris

    `

    ### CODE BLOCKS

    `This is a normal paragraph:

        This is a code block

    `

    ### HORIZONTAL RULES

    `* * *

    ***

    *****

    - - -

    --------------------------------------

    `

    ### LINKS

    `This is [an example](http://example.com/ "Title") inline link.

    [This link](http://example.net/) has no title attribute

    `

    ### EMPHASIS

    `*single asterisks*

    _single underscores_

    **double asterisks**

    __double underscores_

    `

    ### CODE

    `Use the `printf()` function

    `

    ### AUTOMATIC LINKS

    `&lt;http://example.com/&gt;
